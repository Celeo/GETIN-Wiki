# GETIN-Wiki

Eve-authenticated wiki app.

## Setup

1. Clone the repo
1. Install dependencies with `yarn` or `npm i`
1. Run the server server with `yarn dev` or `npm run dev`

## Development

```bash
$ yarn dev
```

Open your browser to `http://localhost:8080`.

## Production

```bash
$ yarn build
```

Transpiled and bundled JS is in ./dist/.
